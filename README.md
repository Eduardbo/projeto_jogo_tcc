# Projeto Jogo TCC

Este repositório contém o projeto de um jogo desenvolvido no **GameMaker** como parte de estudos em desenvolvimento de jogos e programação.

## Como baixar

Clone o repositório usando Git:

```
git clone https://github.com/Eduardbo/projeto_jogo_tcc.git
```

Depois entre na pasta do projeto:

```
cd projeto_jogo_tcc
```

Abra o arquivo `Druida.yyp` no GameMaker para executar ou editar o projeto.
